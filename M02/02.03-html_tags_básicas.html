Tag <h1></h1>

Existem do <h1> até <h6>. São usados para dá títulos e cabeçario na página.

Tag <p></p>

Usado para escrever textos e parágrafos

Tag <a href=""></a>

Usado para linkar. Ela acompanha um atributo (href="") usado para linkar um endereço da internet

Exemplo:

  • <a href="https://www.google.com.br">Ir para o site do Google</a>

  • Ir para o Site do Gogle

Tag <img src="" alt="">

Usado para adiciona uma imagem. Acompanha 2 atributos (src="") para linkar a imagem e (alt="") para uma frase de alternativa caso a imagem não apareça.

Exemplo:

  • <img src="https://www.google.com.br/google.jpg" alt="Logo do Google">

Tag <button> e </button>

Usado para inserir um botão clicável na página.

Exemplo:

  • <button>Clique em Mim!</button>

Tag <br>

Usado para pular uma linha

Tag <ul><li></li></ul>

Usado para uma lista desordenada.

Lista de Bolo [Desordenada]

  • Farinha
  • Ovo
  • Áçucar
  • Leite

Tag <ol><li></li></ol>

Usado para uma lista ordenada.

Lista de Bolo [Ordenada]

  1. Farinha
  2. Ovo
  3. Áçucar
  4. Leite
