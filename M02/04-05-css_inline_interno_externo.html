CSS - Estilo Inline

Style Inline é quando inserimos o código na própria linha de código.

CSS - Estilo Interno

Style Interno é quando inserimos o código no cabeçario <head></head>.

CSS - Estilo Externo

Style Externo é criado uma arquivo .css fora do .html. E para que o arquivo .html saiba desse arquivo exista, precisamos linkar esse .css no <head></head> do arquivos .html.
