Webkit

Prof° explica sobre Webkit.

Webkit é uma engine (máquina) de renderização do css e html nos navegadores.

Cada navegador tem sua máquina de renderização do css e html e pode ser comporta diferente para lê os códigos. Segue os atuais:

  • Edge: EdgeHTML
  • Firefox: Gecko
  • Opera: Webkit
  • Safari: Webkit
  • Chrome: Webkit (Blink)
