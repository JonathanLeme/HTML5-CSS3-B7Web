Existe 3 tipos de colocar estilos:

1° Pelas próprias TAG: "h1, h2, a, p, e etc".

2° Pelo atributo "id"

O atributo "id" é chamado com o "#" no CSS.

O atributo "id" só pode ser usado 1x com a mesma titularidade.

Pelo atributo "class"

O atributo "class" é chamado com "." no CSS.

O atributo "class" pode ser usado várias vezes com a mesma titularidade.
